# ME-portfolio

Prototype portfolio site scaffold.


Preview locally:

1. Open `index.html` in a browser (double-click) or run a simple file server.

2. For a local server (recommended) run in PowerShell:

```
# from project root
python -m http.server 8000

# then open http://localhost:8000
```

Pages:

- `index.html` — Home / About
- `projects.html` — Projects listing and filters
- `contact.html` — Contact form and CTA

Design notes:
- Color palette: #121212, #818b7e, #a74d4a, #eae6e1, #f5f5f5
- Hero with animated reveal, projects grid with client-side filters, contact CTA
